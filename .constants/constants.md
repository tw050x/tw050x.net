# Constants

Below is a list of all required constants within the platform:

- `database.account.name`: The name of the account database.
- `database.account-billing-collection.name`: The name of the account billing collection in the database.
- `database.account-invitations-collection.name`: The name of the account invitations collection in the database.
- `database.account-memberships-collection.name`: The name of the account memberships collection in the database.
- `database.account-profiles-collection.name`: The name of the account profiles collection in the database.
- `database.assignments.name`: The name of the assignments database.
- `database.assignments-tasks-collection.name`: The name of the assignments tasks collection in the database.
- `database.assignments-tasks-templates-collection.name`: The name of the assignments tasks templates collection in the database.
- `database.authentication.name`: The name of the authentication database.
- `database.authentication-nonces-collection.name`: The name of the authentication nonces collection in the database.
- `database.authorisation.name`: The name of the authorisation database.
- `database.authorisation-permissions-collection.name`: The name of the authorisation permissions collection
- `database.payments.name`: The name of the payments database.
- `database.payments-nonces-collection.name`: The name of the payments nonces collection in the database.
- `database.sessions.name`: The name of the sessions database.
- `database.sessions-activity-collection.name`: The name of the sessions activity collection in the database.
- `database.sessions-logins-collection.name`: The name of the sessions logins collection in the database.
- `database.users.name`: The name of the users database.
- `database.users-credentials-collection.name`: The name of the users credentials collection in the database.
- `database.users-nonces-collection.name`: The name of the users nonces collection in the database.
- `database.users-permissions-collection.name`: The name of the users permissions collection in the database.
- `database.users-profiles-collection.name`: The name of the users profiles collection in the database.
- `service.sessions.event-queue-name`: The name of the sessions event queue.
- `service.users.event-queue-name`: The name of the users event queue.
